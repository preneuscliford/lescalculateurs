Politique de confidentialite

Derniere mise a jour : 2 mars 2026.

Les Calculateurs est un site de simulation et de contenus pedagogiques. Cette page decrit les donnees traitees selon le fonctionnement actuel du site.

1) Donnees traitees automatiquement

Quand vous consultez le site, des donnees techniques peuvent etre traitees par des services tiers : adresse IP tronquee ou pseudonymisee selon leurs regles, type de navigateur, terminal, pages vues, interactions publicitaires et mesures d'audience.

2) Services tiers utilises

  • Google AdSense (diffusion et mesure publicitaire) - ID editeur : ca-pub-2209781252231399.
  • Google Tag Manager - conteneur : GTM-TPFZCGX5.
  • Google Analytics 4 - mesure d'audience : G-2HNTGCYQ1X.
  • Google Funding Choices (CMP) pour la gestion du consentement en France et en EEE.

Ces services peuvent utiliser leurs propres cookies/identifiants et traiter des donnees selon leurs politiques.

3) Stockage local dans le navigateur

Le site utilise aussi le stockage local (localStorage) pour ameliorer l'experience (comparaison de scenarios, historique limite, protections anti-spam, etat de certains formulaires).

4) Observatoire communautaire (fonctionnalite anonyme)

Sur certaines pages, un formulaire de retour d'experience peut etre propose. Les donnees envoyees sont pseudonymisees et stockees via Supabase : hash de profil, type de simulateur, reponses au questionnaire, metadonnees anti-spam (fingerprint, horodatage de soumission, controles techniques).

Aucun compte utilisateur n'est requis pour cette fonctionnalite.

5) Donnees que nous ne demandons pas dans le parcours standard

Le site ne demande pas de creation de compte, ni de piece d'identite, ni de donnees bancaires pour utiliser les simulateurs.

6) Contact

Si vous nous ecrivez a contact@lescalculateurs.fr, les informations que vous transmettez volontairement (email, contenu du message) sont traitees uniquement pour repondre a votre demande.

7) Vos choix

Vous pouvez limiter les cookies et traceurs via les parametres de votre navigateur, et gerer la personnalisation publicitaire dans vos reglages Google. Pour toute question : contact.
