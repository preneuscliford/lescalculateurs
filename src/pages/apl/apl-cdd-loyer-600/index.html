Hypothèses importantes

Cette estimation repose sur un scénario simplifié et sur des hypothèses moyennes. Avant toute démarche, il est conseillé de vérifier les paramètres qui influencent réellement le calcul.

  • Verifier la moyenne de revenus si vous avez des mois incomplets.
  • Comparer avec un scénario chomage si votre contrat se termine.
  • Tester un logement a 550 EUR puis a 650 EUR.

Dans certains cas, une colocation, un logement HLM ou un autre type de bail peut modifier sensiblement le montant.
