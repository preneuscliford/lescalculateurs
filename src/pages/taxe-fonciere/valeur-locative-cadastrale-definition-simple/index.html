Réponse rapide

La valeur locative cadastrale est une base fiscale utilisée pour calculer la taxe foncière et, dans certains cas, la taxe d’habitation. Elle correspond au loyer annuel théorique que pourrait produire le bien s’il était loué.

Ce qu’il faut vérifier

  • La valeur locative cadastrale est une base fiscale utilisée pour calculer la taxe foncière et, dans certains cas, la taxe d’habitation. Elle correspond au loyer annuel théorique que pourrait produire le bien s’il était loué
  • Cette valeur ne reflète pas le prix réel du marché immobilier. Elle dépend de critères administratifs : surface, confort, situation, année de construction
