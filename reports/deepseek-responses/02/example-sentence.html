Dans l'Aisne, les frais notariaux pour un bien à 1600 €/m² représentent un poste important. À Laon, l'acquisition d'une maison de 100 m² engendre des émoluments proportionnels calculés sur la valeur vénale. Les débours pour une transaction à Saint-Quentin incluent les droits d'enregistrement et les garanties. À Château-Thierry, les frais fixes s'appliquent quel que soit le type de patrimoine.

La base de calcul intègre le prix de vente déclaré dans l'acte authentique. Les formalités cadastrales suivent les mêmes procédures sur tout le territoire départemental. Le notaire perçoit sa rémunération selon le barème réglementé en vigueur.
